Supported Boards

This code base already had supported for a handful of boards. However due to my limited collection, only bold ones are (probably) guaranteed to run out of the box. Other are merely implemented based on their schematics and needed help from you to make it work.

NXP MCU

  • LPCXpresso 11u14 with base board (for some peripherals to work)
  • Microbuilder RF1GHZNODE equipped with LPC11u37
  • LPCXpresso 1347 with base board (for some peripherals to work)
  • LPCXpresso 1769 with base board (for some peripherals to work)
  • Embedded Artists LPC4357 OEM & Base board
  • NGX LPC4330 Explorer
  • Keil MCB4357 Evaluation Board

Add your own board

If you don't possess any of supported board above. Don't worry you can easily implemented your own one by following this guide as long as the mcu is supported.
